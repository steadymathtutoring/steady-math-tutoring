# Steady Math Tutoring

A simple website for promoting university-level calculus tutoring.  
It includes service information, tutor bio, testimonials, and contact details.

## Features
- Clean and simple landing page (`index.html`)
- Custom logo for Steady Math Tutoring
- Mobile-friendly layout
- Easy to update and deploy via GitHub Pages

## About the Tutor
Nour is a mathematics tutor with a strong background in university calculus.  
With a degree in mathematics and several years of experience, Nour specializes in helping students understand limits, derivatives, integrals, and other core concepts.
